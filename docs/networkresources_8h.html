Creates a bound UDP socket, taking the host name/IP address and service name/port number as arguments.

The arguments passed to this function are used verbatim for a getaddrinfo() call, and the same rules apply.

Setting node to NULL will allow binding to the wiledcard address.

Function wrapping the recvfrom() network socket call.

This function is primarily useful for unconnected sockets, such as a server-side UDP socket or peer-to-peer UDP socket.

Connected UDP sockets, and other kinds of network sockets (including TCP client sockets), should use the generic read/write functions, and can safely be incorporated into composite resources as a consequence.
