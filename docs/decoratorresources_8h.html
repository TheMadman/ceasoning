Constructor for csalt_resource_decorator_logger, taking similar arguments to csalt_store_decorator_logger().

This decorator also supports setting a message for the csalt_resource_init() function.

The resource only checks for error or success states on csalt_resource_init. If the resource is initialized successfully, a pointer to a csalt_store_decorator_logger is passed, which has all the message lists passed to it, enabling logging on the resulting store as well.
