Detailed Description

Provides a wrapper around on-disk files for use with csalt_resource algorithms.

This file provides a concrete file type, based on file descriptors, to on-disk files specified by a string pathname.

Allows the creation of new files and opening of existing files.

Creates a file resource with the given path and flags.

The flags are those passed to unistd.h's open function.

Some flags interfere with normal operation of the API, specifically:

  • O_APPEND

Others will cause some functions to error, such as opening a file read-only causing csalt_resource_file_write to error. The file resource will still be usable with the features it does support.

Some flags are implicit with the expected functionality of the library, specifically:

  • O_NONBLOCK

These need not be specified and are set automatically.
