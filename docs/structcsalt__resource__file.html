Detailed Description

Struct representing a file resource supporting lazy-loading.

Do not attempt to initialize directly; instead, use the csalt_resource_file function.

Files opened with this library are automatically set to non-blocking, and if a read/write operation would block, the csalt read/write functions simply return 0 bytes read/written. This allows the generic read/write functions to separate zero bytes transferred from an error arising from the resource itself.
