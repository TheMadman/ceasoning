Detailed Description

Converts byte-indexing into array-like-indexing.

If your store's data represents an array of fixed-length elements, this decorator scales the traditional "bytes" argument by the size of each element automatically, similarly to the C array index operator square brackets.

For example, for an array of ints which are 32 bits (4 bytes) long, a csalt_store_read(store, buffer, 1) will attempt to read 4 bytes; a csalt_store_split(store, 3, 5, block, arg) will split beginning at the 3rd int (12 bytes from the beginning) and ending at the 5th int (20 bytes from the beginning), and so on.

csalt_store_size() returns the original store's size divided by the element size.
