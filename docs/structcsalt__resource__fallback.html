Detailed Description

This type provides a resource which initializes multiple resources, returning a csalt_store_fallback on initialization.

This resource attempts to initialize all resources, returning 0 if any of them fail and a pointer to a csalt_store_fallback on success. A constructor is available taking two resources, and one is available for an array of resources.
