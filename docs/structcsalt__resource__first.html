Detailed Description

A resource which returns the store of the first resource to initialize successfully.

On csalt_resource_init(), the first resource is tried. If the first resource initializes successfully, its store is returned without initializing the second resource.

If the first resource fails to initialize, the second resource is attempted. On success, its store is returned. Otherwise, NULL is returned.
