Detailed Description

Decorates a resource with a logger, reporting resource initialization failures.

If the initialization succeeds, the resulting store is also wrapped in a csalt_store_logger.

Constructs a csalt_resource_logger.

If you want to log the success/failure of the resource initialization, you should provide a csalt_log_message for csalt_resource_init().

To log messages for the store operations, simply add them to the array as you would for csalt_store_logger.

Parameters
