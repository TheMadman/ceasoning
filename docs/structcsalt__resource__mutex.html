Detailed Description

Decorates a resource for synchronization with a mutex.

csalt_resource_init() attempts to initialize both the resource and the mutex. If either fails, NULL is returned. Otherwise, a pointer to a csalt_store_mutex is returned, wrapping the initialized resource.
